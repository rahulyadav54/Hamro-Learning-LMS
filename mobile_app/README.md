# Hamro Learning Mobile App

Starter Flutter app structure for the Hamro Learning LMS.

## What this includes

- Student-focused navigation
- Authentication shell
- Dashboard, courses, payments, and profile screens
- API service placeholders for the Laravel backend
- Theme and shared UI components

## Next step

Run `flutter pub get` after creating the Flutter project files, then connect the API base URL to your Laravel backend.

## Backend routes used

- `POST /api/login`
- `GET /api/user`
- `GET /api/my-courses`
- `GET /api/dashboard`
- `GET /api/live-classes`
- `GET /api/purchases`
